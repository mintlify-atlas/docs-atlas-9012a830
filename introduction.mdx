## What is PX47?

PX47 is a **production-ready backend data-processing pipeline** for audio and video files, built with Express.js. It leverages direct S3 uploads via pre-signed URLs and processes files asynchronously using BullMQ workers with Redis.

## Key Features

<AccordionGroup>
  <Accordion title="Direct S3 Uploads" icon="cloud-arrow-up">
    Upload files directly to S3 using pre-signed URLs, bypassing your backend server for maximum performance and reduced bandwidth costs.
  </Accordion>

  <Accordion title="Asynchronous Processing" icon="clock">
    BullMQ workers process files in the background with Redis-backed queues, allowing instant API responses and horizontal scalability.
  </Accordion>

  <Accordion title="Audio Processing" icon="waveform">
    - Format conversion (MP3/AAC with configurable bitrate)
    - Metadata extraction using FFprobe
    - Waveform JSON generation with audiowaveform
    - All processed files uploaded back to S3
  </Accordion>

  <Accordion title="Video Processing" icon="video">
    - Automatic audio track extraction
    - Mute video generation
    - Comprehensive metadata extraction (codec, resolution, fps, color space)
    - Separate audio and video file uploads to S3
  </Accordion>
</AccordionGroup>

## Use Cases

<Note>
  PX47 is perfect for applications that need to process user-uploaded media files without blocking the main request-response cycle.
</Note>

- **Podcast platforms**: Convert uploads to standardized formats with waveform visualization
- **Video streaming services**: Extract audio tracks and generate mute previews
- **Content management systems**: Process media files asynchronously with metadata extraction
- **Social media platforms**: Handle large-scale media uploads with distributed workers
